Personalización

En esta sección puedes personalizar la apariencia de la aplicación según tus preferencias.

Opciones:

  • Escoge el color del texto

Pasos:

  1. Haz clic en la opción Escoge el color del texto.
  2. Se abrirá un cuadro de diálogo JColorChooser donde podrás elegir el color para el texto.
  3. Selecciona el color para los ingredientes y luego elige otro color para los postres, si lo deseas.
  4. El color seleccionado se aplicará automáticamente a los textos en la aplicación.
